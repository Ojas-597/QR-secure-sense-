QR Secure Sense is an interactive cybersecurity awareness project designed for exhibitions, colleges, and workshops.

The system shows a **fake malware alert** after scanning a QR code (purely simulated and safe), then guides users through:

##  Features

### **1. QR Scan Simulation**
Visitors scan a QR code ‚Üí are taken to a dynamic ‚Äúmalware detected‚Äù simulation page.

### **2. Fake Malware Demonstration (SAFE)**

- Fake warnings

### **3. Device Self-Assessment**
The app guides the visitor to:
- Check app permissions  
- Check unknown app installs  
- Review network safety  
- Identify suspicious behaviours  

### **4. Malware Removal Simulation**
A ‚ÄúCleanup Wizard‚Äù explains:

### **5. Awareness Learning + Quiz**

- Short cyber safety lessons  
- MCQ quiz (options A, B, C, D)

### **6. Admin Dashboard**
Protected login for exhibitors:

- View scan events  
- Reset demo  
- Manage awareness questions
